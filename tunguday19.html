Tài liệu học từ vựng Ngày 19

Các mẫu ngữ pháp được sử dụng trong Ngày 19:

  1. 〜では・には (de wa / ni wa): Ở (đâu đó) thì.../ đến (đâu đó) thì...
    Công thức: N + では・には
    Ví dụ: そこへは何回も行った。(Soko e wa nankai mo itta. - Chỗ đó thì tôi đến mấy lần rồi.)
  2. Thể sai khiến (shieki kei): Bắt/ cho phép/ làm cho...
    Công thức: N1 に N2 (を) V (thể sai khiến)
    Ví dụ: 社長は、ちこくが多い社員をやめさせました。(Shachou wa, chikoku ga ooi shain o yamesasemashita. - Giám đốc đã buộc thôi việc những nhân viên đi làm muộn nhiều.)
  3. Thể sai khiến bị động (shieki shieki hidou kei): Bị bắt/ được cho phép...
    Công thức: N1 は N2 に (N を) V (thể sai khiến bị động)
    Ví dụ: 子供のころ、毎日母に野菜を食べさせられました。(Kodomo no koro, mainichi haha ni yasai o tabesaseraremashita. - Hồi còn nhỏ, tôi bị mẹ bắt ăn rau mỗi ngày.)
  4. 〜させてください (sasete kudasai): Hãy cho tôi...
    Công thức: V (thể sai khiến) て + ください
    Ví dụ: この仕事をさせてください。(Kono shigoto o sasete kudasai. - Hãy cho tôi làm công việc này nhé.)
  5. Động từ cho - nhận (kính ngữ) (ageru morau kuremasu - keigo):
    もらう → いただく (morau → itadaku) : nhận (khiêm nhường)
    あげる → さしあげる (ageru → sashiageru) : cho, tặng (khiêm nhường)
    くれる → くださる (kureru → kudasaru) : cho, tặng (tôn kính)
  6. Tôn kính ngữ (sonkei go):
    います → いらっしゃいます・おいでになります (imasu → irasshaimasu / oide ni narimasu)
    来ます・行きます → いらっしゃいます・おいでになります (kimasu / ikimasu → irasshaimasu / oide ni narumasu)
    します → なさいます (shimasu → nasaimasu)
    言います → おっしゃいます (iimasu → osshaimasu)
    食べます・飲みます → 召し上がります (tabemasu / nomimasu → meshiagarimasu)
    見ます → ご覧になります (mimasu → goran ni narimasu)
    知っています → ご存じです・ご存じでいらっしゃいます (shitte imasu → gozonji desu / gozonji de irasshaimasu)
    死にます → お亡くなりになります (shinimasu → onakunari ni narimasu)
    くれます → ください (kuremasu → kudasai)
    いいです → よろしいです (ii desu → yoroshii desu)
  7. Khiêm nhường ngữ (kenjou go):
    〜です → 〜でございます (desu → de gozaimasu)
    あります → ございます (arimasu → gozaimasu)
    来ます・行きます → 参ります・伺います (kimasu / ikimasu → mairimasu / ukagaimasu)
    います → おります (imasu → orimasu)
    します → いたします (shimasu → itashimasu)
    言います → 申します・申し上げます (iimasu → moushimasu / moushiagemasu)
    食べます・飲みます → いただきます (tabemasu / nomimasu → itadakimasu)
    訪ねます → 伺います・お邪魔します (tazunemasu → ukagaimasu / ojama shimasu)
    聞きます → 伺います (kikimasu → ukagaimasu)
    見ます → 拝見します (mimasu → haiken shimasu)
    知っています → 存じます・存じております (shitte imasu → zonjimasu / zonjite orimasu)
    知りません → 存じません (shirimasen → zonjimasen)
    会います → お目にかかります (aimasu → ome ni kakarimasu)
    死にます → 亡くなります (shinimasu → nakunarimasu)
    あげます → 差し上げます (agemasu → sashiagemasu)
    もらいます → いただきます (moraimasu → itadakimasu)
    いいです → 結構です (ii desu → kekkou desu)
